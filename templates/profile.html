Two-Factor Authentication Help
What is Two-Factor Authentication (2FA)?

Two-factor authentication adds an extra layer of security to your account by requiring a second form of verification in addition to your password.

How does it work?
  • Setup: Scan a QR code with an authenticator app (like Google Authenticator, Authy, or Microsoft Authenticator)
  • Login: Enter your password, then provide a 6-digit code from your authenticator app
  • Backup Codes: Save 10 backup codes for emergency access if you lose your device
Security Benefits
  • Protects against password theft
  • Prevents unauthorized access even if your password is compromised
  • Meets security requirements for sensitive data
Getting Started
  1. Click "Enable 2FA" to start the setup process
  2. Download an authenticator app if you don't have one
  3. Scan the QR code with your authenticator app
  4. Enter the 6-digit code to verify setup
  5. Save your backup codes in a secure location
Tip: Keep your backup codes in a safe place. You'll need them if you lose access to your authenticator app.
